◆ toString()

synchronized String selogger.logging.io.LatestEventBuffer.toString ( )
inline

Generate a string representation that is written to a trace file.

戻り値
A line of CSV string. The first column is the number of events recorded in the buffer. The other columns are the event data recorded in a trace. The oldest event is written first. the latest one is written at last. For each event, the observed value, the sequence number, and the thread ID are written. In case of a string object, the content is written with the object ID.
