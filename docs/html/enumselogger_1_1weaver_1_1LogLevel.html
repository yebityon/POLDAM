◆ IgnoreArrayInitializer

selogger.weaver.LogLevel.IgnoreArrayInitializer

Ignore instructions that assign initial values to an array. For example, an array creation statement "int[] a = new int[]{1, 2, 3};" is compiled to three instructions a[0]=1, a[1]=2, a[2]=3. SELogger inserts logging code for the three ARRAY_STORE instructions. This kind of code may be unacceptable if the method initializes a large array object.

LogLevel.java の 20 行目に定義があります。
